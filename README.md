AI Exam Strategy Generator with Quiz System

Overview: This web application helps students plan their studies
effectively and test their understanding using AI.

Features: - Smart Study Plan Generator - Unit-based daily scheduling -
High priority subject detection - AI-powered quiz generation (Google
Gemini API) - Self-evaluation with score and analysis - Responsive
modern UI

How It Works: 1. User enters subjects and units 2. App calculates
available study days 3. Units are distributed across days 4. Each day
shows what to study 5. User can take AI quiz for each day 6. App
evaluates performance and shows weak areas

Tech Stack: - React (Vite) - Tailwind CSS - Google Gemini API - Vercel
(Deployment)

Run Locally: 
1. npm install 
2. Create .env file and 
add:GEMINI_API_KEY=your_api_key_here 
3. npm run dev

Live Demo: (https://ai-exam-planner-with-quiz-system.vercel.app/)
